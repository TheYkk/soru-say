# Nasil Calisitirilir

**Not** projenin calismasi icin node.js v4.x.x ve ya daha yuksek  ve npm 3.x.x. gereklidir

```bash
git clone https://github.com/TheYkk/soru-say
cd soru-say
npm i   
npm start    # uygulamayi baslatir
```
